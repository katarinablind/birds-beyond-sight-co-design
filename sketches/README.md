# Sketch photos for non-tech ideations

Drop a photo of a hand sketch here (e.g. `trailhead-post.jpg`), then reference it
on the matching registry entry in `playground.html`:

    {id:'sketch-…', kind:'sketch', img:'sketches/trailhead-post.jpg', note:'…', …}

Only the sticky-note text is editable in the tool; the photo always comes from this folder.
